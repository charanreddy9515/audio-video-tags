Basic Audio and Video Embedding

This page demonstrates the use of the HTML5 < audio> and < video> tags to embed media directly into a web page. The controls for playback, volume, etc., are provided by the browser's default implementation.

Audio Example

This is an audio player. The controls attribute is used to display the playback controls. You can replace the src URL with your own audio file.

This is a video player. The controls attribute provide the play, pause and volume controls. You can replace the src URL with your own video file.
